  Senior Full Stack Engineer at **DEPT®** with some years delivering production systems end-to-end — TypeScript/NestJS backends, React/Next.js frontends, and cloud-native infrastructure on AWS with Docker and Terraform. Track record across diverse industries: sports media platforms, large-scale e-commerce, insurance, and world-wide-scale ticketing
  systems.

  Lately focused on agentic AI: built a QA automation agent (Claude Agent SDK + Qdrant + Playwright) that ingests Jira tickets, generates test cases, runs them against a live app, and files deduplicated bug reports autonomously.
